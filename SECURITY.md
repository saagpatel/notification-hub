# Security Policy

## Supported Versions

Security updates are applied to the default branch. This repository does not publish long-term support branches yet.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately through GitHub Security Advisories when available. If advisories are not available, open a minimal private report with reproduction context and avoid posting secrets or exploit details in public issues.

Include:

- affected component or endpoint
- steps to reproduce
- observed impact
- suggested mitigation, if known

I will review reports as quickly as possible and prioritize fixes based on exposure and exploitability.
